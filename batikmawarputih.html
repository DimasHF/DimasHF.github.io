Ecommerce Batik Store

This project is an online store for a local batik brand, built to help them sell their products digitally and reach more customers. The system supports everything from browsing products to checking shipping costs and completing payments. My main job was to build the backend API that powers all of that.

What I did:

  • Developed a RESTful API using Laravel to handle product listings, categories, orders, payments, and user authentication.
  • Integrated with RajaOngkir API to calculate real-time shipping costs based on customer address and selected courier.
  • Connected to Tripay payment gateway, enabling customers to pay using virtual accounts, e-wallets, or QRIS.
  • Structured the database to manage products, stock, transactions, and shipping status smoothly.
  • Secured the API using Sanctum for authentication and role-based access for admin and customers.

Challenges & How I Solved Them:

  • Shipping calculation needed to handle multiple couriers and different service types.
  • Solution: Solved by building a dynamic wrapper for the RajaOngkir API, making it reusable across the platform.
  • Payment status updates needed to be handled in real-time.
  • Solution: Implemented Tripay webhook listener to update payment status automatically and activate orders only when payment is confirmed.
